# Development
Get a `credentials.json` file: [see here](https://developers.google.com/docs/api/quickstart/python).
Then dump it into the StatsimusPrime folder here.

```bash
 $ python3 -m venv venv
 $ source venv/bin/activate
 (venv) $ pip install -r requirements.txt
```
